chseets Public API

Access our citysheet and cheat sheet database programmatically. The API is free, open, and requires no authentication.

Base URL: https://chseets.com/api

Use case: Quest Hub & Content Base for WASD Travel Game Master AI

Static JSON Endpoints (Recommended)

These endpoints serve pre-generated JSON files. Fast, reliable, no API limits.

GET /api/meta.json

API version and metadata.

curl https://chseets.com/api/meta.json

GET /api/sheets.json

List all published citysheets.

curl https://chseets.com/api/sheets.json

GET /api/sheets/{slug}.json

Fetch a single citysheet with full POI data.

curl https://chseets.com/api/sheets/weiz.json
curl https://chseets.com/api/sheets/thannhausen.json

Supabase REST API (Live Database)

Direct access to the Supabase database. Requires API key header.

curl "https://zpvqcjrtsbznaaphjhug.supabase.co/rest/v1/sheets_meta?select=*" \
  -H "apikey: YOUR_ANON_KEY"

Code Examples

JavaScript (fetch)

// List all citysheets
fetch('https://chseets.com/api/sheets.json')
  .then(res => res.json())
  .then(data => console.log(data.results))
  .catch(err => console.error(err));

// Get Weiz citysheet with POIs
fetch('https://chseets.com/api/sheets/weiz.json')
  .then(res => res.json())
  .then(sheet => {
    console.log(sheet.title);      // "Weiz Citysheet"
    console.log(sheet.pois);       // Array of 5 POIs
    console.log(sheet.pois[0].story); // Quest prompt
  });

// Get Thannhausen citysheet
fetch('https://chseets.com/api/sheets/thannhausen.json')
  .then(res => res.json())
  .then(sheet => console.log(sheet.pois));

Python (requests)

import requests

# List all citysheets
response = requests.get('https://chseets.com/api/sheets.json')
sheets = response.json()['results']
print(f"Found {len(sheets)} citysheets")

# Get Weiz citysheet
response = requests.get('https://chseets.com/api/sheets/weiz.json')
weiz = response.json()
for poi in weiz['pois']:
    print(f"{poi['id']}: {poi['name']} - {poi['story']}")

Pricing and Limits

The chseets API is completely free:

  • Static JSON: No rate limits, cached by CDN
  • CORS: Enabled for all origins
  • Authentication: Not required
  • Cost: Free for all users

For Developers

The OpenAPI specification is available at docs/api/openapi.yaml in the repository.

  • Describes all endpoints, parameters, and response formats
  • Load it into OpenAPI tools to explore the service
  • Generate client SDKs for your preferred language
