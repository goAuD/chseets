Public API (Preview)

The chseets API will allow external tools and platforms to list, search, and embed sheet data directly.

Planned endpoints

  • GET /api/sheets - list all available sheets
  • GET /api/sheets/{slug} - get specific sheet metadata
  • POST /api/search - contextual search by keyword

Auth & Rate-limits

Public read-only access will require no token. Extended use (custom apps, analytics) will use simple API keys with fair limits.

For early testing access, reach out at chseets@wasdtravel.com.
