chseets Public API

Access our cheat sheet database programmatically. The API is free, open, and requires no authentication.

Base URL: https://chseets.com/api

Endpoints

GET /api/meta

API version and metadata.

curl https://chseets.com/api/meta

GET /api/sheets

List all published sheets. Supports optional ?page= and ?limit= query params.

curl https://chseets.com/api/sheets

GET /api/sheets/{slug}

Fetch a single sheet by its URL slug.

curl https://chseets.com/api/sheets/javascript-basics

POST /api/search

Search sheets by keyword. Send JSON body with q field.

curl -X POST https://chseets.com/api/search \
  -H "Content-Type: application/json" \
  -d '{"q": "java"}'

Code Examples

JavaScript (fetch)

// List all sheets
fetch('https://chseets.com/api/sheets')
  .then(res => res.json())
  .then(data => console.log(data.results))
  .catch(err => console.error(err));

// Get single sheet
fetch('https://chseets.com/api/sheets/javascript-basics')
  .then(res => res.json())
  .then(sheet => console.log(sheet));

// Search
fetch('https://chseets.com/api/search', {
  method: 'POST',
  headers: { 'Content-Type': 'application/json' },
  body: JSON.stringify({ q: 'python' })
})
  .then(res => res.json())
  .then(data => console.log(data.results));

Python (requests)

import requests

# List all sheets
response = requests.get('https://chseets.com/api/sheets')
print(response.json())

# Get single sheet
response = requests.get('https://chseets.com/api/sheets/javascript-basics')
print(response.json())

# Search
response = requests.post(
    'https://chseets.com/api/search',
    json={'q': 'linux'}
)
print(response.json())

Response Format

All responses are JSON. Example sheet object:

{
  "title": "JavaScript Basics",
  "slug": "javascript-basics",
  "language": "en",
  "tags": ["javascript", "programming", "web"],
  "summary": "Quick reference for JS fundamentals",
  "updated_at": "2025-01-15"
}

Pricing and Limits

The chseets API is completely free:

  • Rate limit: 60 requests per minute per IP
  • CORS: Enabled for all origins
  • Authentication: Not required
  • Cost: Free for all users

For Developers

The OpenAPI specification is available at docs/api/openapi.yaml in the repository.

  • Describes all endpoints, parameters, and response formats
  • Load it into OpenAPI tools to explore the service
  • Generate client SDKs for your preferred language
