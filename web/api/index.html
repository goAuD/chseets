Available Endpoints

GET /sheets_meta

List all published sheets. Returns metadata for all publicly available cheat sheets.

GET /rest/v1/sheets_meta?select=*&published=eq.true&limit=50

GET /sheets_meta?slug=eq.{slug}

Get single sheet. Fetch a specific sheet by its URL slug.

GET /rest/v1/sheets_meta?slug=eq.javascript-basics&select=*

GET /sheets_meta?title=ilike.*keyword*

Search sheets by title (case insensitive).

GET /rest/v1/sheets_meta?title=ilike.*python*&published=eq.true

Code Examples

JavaScript (fetch)

const API_URL = 'https://zpvqcjrtsbznaaphjhug.supabase.co/rest/v1';
const API_KEY = 'eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9...'; // Use full key

fetch(`${API_URL}/sheets_meta?select=*&published=eq.true&limit=10`, {
  headers: {
    'apikey': API_KEY,
    'Content-Type': 'application/json'
  }
})
.then(res => res.json())
.then(data => console.log(data))
.catch(err => console.error(err));

Pricing and Limits

The chseets API is completely free. Powered by Supabase free tier:

  • CORS: Enabled for all origins
  • Authentication: Not required for public sheets
  • Cost: Free for all users, no hidden fees
