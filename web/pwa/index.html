Install chseets as an App

chseets is a Progressive Web App (PWA). You can install it on your device for:

  • Instant access from your home screen or desktop
  • Offline viewing of cached sheets
  • Faster loading with local caching
  • Full screen experience without browser UI

Chrome, Edge, Brave (Desktop and Android)

  1. Visit chseets.com
  2. Look for the install icon in the address bar
  3. Click it and select "Install"
  4. The app will appear in your Start Menu or Applications folder

On Android: Tap the menu and select "Add to Home screen" or "Install app"

Firefox (Desktop and Android)

Firefox does not support automatic PWA install prompts. Use these steps:

  1. Visit chseets.com
  2. Click the address bar and look for the home icon
  3. Click "Add to Home screen" (on Android)
  4. On desktop: Bookmark the page or use a PWA extension

Alternatively, install the NicePWA extension for Firefox.

Safari (iOS and macOS)

  1. Open chseets.com in Safari
  2. Tap the Share button
  3. Scroll down and tap "Add to Home Screen"
  4. Name it and tap "Add"

On macOS Sonoma+: File then "Add to Dock" or use the share menu.

Troubleshooting

Install banner not showing?

  • Ad blockers (uBlock Origin, AdBlock, etc.) may hide the install prompt. Temporarily disable them or add chseets to your whitelist.
  • Already dismissed? Clear your browser's site data or try an incognito window.
  • Already installed? Uninstall the app first if you want to reinstall.
